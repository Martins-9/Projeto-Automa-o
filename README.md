Conversor de Temperatura (Celsius → Fahrenheit)

Este projeto consiste em um programa simples que realiza a conversão de temperaturas de Celsius para Fahrenheit. O usuário informa a temperatura em graus Celsius, e o sistema retorna o valor correspondente em Fahrenheit, utilizando a fórmula matemática padrão.

O objetivo do projeto é praticar lógica de programação, entrada e saída de dados e operações matemáticas básicas, sendo ideal para fins educacionais e para compor um portfólio no GitHub.
